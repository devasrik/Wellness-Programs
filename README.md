# Wellness Programs Web App

🔗 Live Demo: https://devasrik.github.io/Wellness-Programs/

A responsive, user-friendly wellness programs website designed to promote healthy living through curated programs and resources. This project demonstrates my ability to design, build, and deploy a modern web application using clean UI principles and front-end best practices.

---

## Features

- Clean and intuitive user interface  
- Fully responsive design (desktop, tablet, and mobile)  
- Well-organized sections for wellness programs and content  
- Deployed using GitHub Pages  
- Consistent styling and layout for a professional look  
- Fast load time and lightweight front-end implementation
